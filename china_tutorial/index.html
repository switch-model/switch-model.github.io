Files for the Switch tutorial

Below you can find files for the Switch-China tutorial. Please download the solvers .zip file for your operating system and the two Switch_Tutorial_*.zip files. Then follow the pre-tutorial instructions to setup your system. After that you will be ready to follow the main tutorial instructions. After the main tutorial, please download the Switch-China assignment and work on that before the second session.
