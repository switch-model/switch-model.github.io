SWITCH Power System Planning Model

This is the home of the SWITCH power system planning model. Documentation and downloads can be found under the switch-model organization page at http://github.com/switch-model.

There you will find the following components:

  • The SWITCH repository: https://github.com/switch-model/switch. This contains the generic SWITCH model, which can be used to create models for new power systems, and is also required to run existing power system models (e.g., the SWITCH-Hawaii repositories). This is also a good place to find general documentation for modeling with SWITCH, which will be helpful if you need to customize any SWITCH-based models.
  • The switch-hawaii-studies repository: https://github.com/switch-model/switch-hawaii-studies. This is the place to start if you want to use the existing SWITCH-Hawaii model. This contains code and data to define all the different Hawaii scenarios that have been run. You will also find instructions for installing the switch and switch-hawaii-core repositories to support the switch-hawaii-studies repository.
  • The switch-hawaii-core repository: https://github.com/switch-model/switch-hawaii-core. This contains code which is common to all the SWITCH-Hawaii scenarios (e.g., to read and write from the switch-hawaii database server). Instructions for installing this are given in the switch-hawaii-studies repository.

Please contact Matthias Fripp at mfripp@hawaii.edu if you have questions or need help installing and running SWITCH.
